# Minh Huy · 2026

Trang cá nhân của Minh Huy — lớp 10, Trường THPT Nguyễn Du, niên khóa 2026–2027.

Tiền phong bóng rổ, 9 năm liền học sinh xuất sắc, Giải Nhì Toán Máy tính cầm
tay cấp Thành phố, và một chú chó tên No Name.

## Chạy thử local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push lên nhánh `main` sẽ tự động build và deploy lên GitHub Pages qua GitHub
Actions (`.github/workflows/deploy.yml`).

Trang chạy tại: `https://mashinhat.github.io/minhhuy2026/`

> Cần bật GitHub Pages trong **Settings → Pages → Source: GitHub Actions**
> ở repo trước khi deploy lần đầu.

## Ảnh thật

Hiện site đang dùng khung ảnh placeholder (icon minh họa). Khi có ảnh thật
của Minh Huy / No Name, thay vào `src/components/DogSection.tsx` và
`src/components/CourtSection.tsx`.
